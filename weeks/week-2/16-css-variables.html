CSS Variables

The var() function is used to insert the value of a CSS variable.

CSS variables have access to the DOM, which means you can create variables with local or global scope, change variables with JavaScript, and change variables based on media queries.

A good way to use CSS variables is when it comes to the colors of your design. Instead of copying and pasting the same colors over and over, you can place them in variables.

How var() works

First: CSS variables can have global or local scope.

Global variables can be accessed/used throughout the document, while local variables can only be used within the selector where they are declared.

To create a variable with global scope, declare it within the :root selector. The :root selector matches the root element of the document.

To create a variable with local scope, declare it within the selector that will use it.
